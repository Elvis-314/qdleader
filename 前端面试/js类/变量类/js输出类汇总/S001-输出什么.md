# 输出什么


## 1.
```js
let a = {},  
b = '0',  
c = 0;  
  
a[b] = '唐诗';  
  
a[c] = '宋词';  
  
console.log(a[b]);

```






## 答案
宋词


## 解析
> js 中对象的 key 只能是 string、symbol
> a[b] 与 a[c] 都可看做是 a['0'], 最后一次赋值是 宋词 所以结果也就是 宋词






## 2. 
```js
let a = {},  
b = {  
    n: '1'  
},  
c = {  
    m: '2'  
};  
  
a[b] = '唐诗';  
  
a[c] = '宋词';  
  
console.log(a[b]);


```





## 答案
宋词


## 解析

会调用对象的 toString 方法, 所以 b、c 都等于 [object Object]
