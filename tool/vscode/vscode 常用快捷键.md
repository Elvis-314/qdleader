

# 通过匹配文本打开文件
Windows: Ctrl + T 
Mac: command + T
说到搜索文件，你可以动态地搜索和打开文件。这是我最喜欢的特性之一，因为不需要手动单击目录来重新打开一个不再打开的文件。


## 打开终端

```js
control + ~
```


### 快速引动到这一行的开头或结尾
command + 左右箭头


### 复制光标向上或者向上批量添加内容
a. Ctrl + Alt +向下箭头
(Mac: command + Option + 向下箭头)将光标添加到下面。


b. command + 左右箭头
